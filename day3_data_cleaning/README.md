# 📊 Data Cleaning & Preprocessing Project

## 🚀 Overview
This project focuses on cleaning and preprocessing a real-world dataset related to customer churn events. The goal is to transform raw, inconsistent data into a structured and analysis-ready format.

---

## 📁 Dataset
- **Raw Data:** `ravenstack_churn_events.csv`
- **Cleaned Data:** `ravenstack_churn_events_cleaned.csv`

---

## 🎯 Objectives
- Handle missing values
- Remove duplicates
- Standardize data formats
- Fix inconsistent entries
- Prepare dataset for analysis/modeling

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- Jupyter Notebook  

---

## 🔹 Key Steps Performed

### 1. Data Exploration
- Checked dataset shape, columns, and data types  
- Identified null values and inconsistencies  

### 2. Data Cleaning
- Handled missing values using appropriate strategies  
- Removed duplicate records  
- Standardized categorical values  

### 3. Data Transformation
- Converted data types where necessary  
- Renamed columns for clarity  
- Formatted date/time fields  

### 4. Final Output
- Exported clean dataset for further analysis  

---

## 📌 Key Functions Used
- `pd.read_csv()`
- `df.head()`, `df.info()`, `df.describe()`
- `df.isnull()`, `df.dropna()`, `df.fillna()`
- `df.drop_duplicates()`
- `df.rename()`
- `df.astype()`
- `df.to_csv()`
